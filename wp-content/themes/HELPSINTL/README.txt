Gifted Theme 1.0
http://fourtencreative.com/gifted

INSTALL: 
1. Upload the theme folder via FTP to your wp-content/themes/ directory.
2. Go to Admin WordPress and select Appearance.
3. Select Gifted WordPress theme.

SIDEBARS:
Sidebars load automatically. Works best in Wordpress 2.8.x with the ability to reuse widgets.
1. Go to Admin section, select Appearance, Widgets.
2. Select appropriately title sidebar and add widgets at will.
